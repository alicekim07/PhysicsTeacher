# PHYSICSTEACHER
AI 기반 물리 강의 자동 생성 파이프라인

---

## 프로젝트 소개

**PHYSICSTEACHER**는
대학교 물리학 강의 슬라이드(PPTX / PDF)를 입력으로 받아,
AI를 활용해 **강의 스크립트 -> 음성 -> 자막 -> 강의 영상**을 자동으로 생성하는 프로젝트입니다.

단순한 콘텐츠 자동화가 아니라,
- AI가 물리 개념을 **어디까지 이해하고 설명할 수 있는지**
- 실제 강의 흐름과 유사한 **설명 맥락을 유지할 수 있는지**
를 실험하는 것을 목표로 합니다.

---

## 프로젝트 목적

- 물리 강의 콘텐츠 제작 과정의 자동화 가능성 탐구
- 슬라이드 기반 설명에서 AI의 개념 이해 한계 분석
- 강의 보조 도구로서 AI 파이프라인 설계 실험

---

## 전체 파이프라인 개요

1. PPTX 파일 입력
2. 강의 노트 및 PDF 변환
3. PDF -> 슬라이드 이미지 변환
4. 슬라이드별 스크립트 생성
5. 음성 합성(TTS)
6. 자막 생성
7. 슬라이드별 영상 생성 및 병합

## 실행 방법

1. 변환할 강의 PPTX 파일을 아래 디렉토리에 넣습니다.
data/pptx/

2. 전체 파이프라인을 실행합니다.

```bash
python run_all.py
```

3. 실행이 완료되면, 강의별 출력 결과가 다음 위치에 생성됩니다.
data/outputs/{lecture_name}/

## 스크립트 생성 설계

본 프로젝트의 스크립트 생성은 단순 요약이나 번역이 아니라,
**실제 강의에서의 설명 흐름을 재현하는 것**을 목표로 설계되었습니다.

각 슬라이드에 대해 다음 단계를 거쳐 스크립트를 생성합니다.

- 슬라이드 이미지로부터 OCR 기반 텍스트 추출
- 슬라이드가 전달하는 핵심 개녈을 **개념 수준에서 분해**
- 교수의 강의 스타일과 설명 맥락을 반영한 설명 지침 생성
- 이전 슬라이드 설명을 요약하여 다음 슬라이드와 **맥락적으로 연결**
- 수식, 기호를 사용하지 않고 말로 풀어 설명하는 강의 스크립트 생성

이를 통해,
- 슬라이드 간 설명 단절을 최소화하고
- 교재 문장처럼 딱딱한 설명이 아닌
- 실제 강의 발화에 가까운 스크립트를 생성하도록 설계하였습니다.

---

## 자막 생성 방식

자막은 음성 인식 결과와 강의 스크립트를 결합하는 방식으로 생성됩니다.

기본 흐름은 다음과 같습니다.

- 음성 파일로부터 자동 음성 인식(ASR)을 통해 타임스탬프 기반 자막 생성
- 생성된 자막의 타이밍을 유지한 채, 실제 강의 스크립트 문장으로 자막 텍스트를 정렬 및 교체

이 방식은,
- 음성 길이에 기반한 자연스러운 자막 타이밍을 유지하면서도
- 강의 스크립트와 자막 내용의 불일치를 최소화하기 위해 선택되었습니다.

자막 분할 및 정렬 과정에서는
문장 길이, 조사 및 구두점 위치를 고려한 휴리스틱을 사용합니다.

## 한계 및 주의사항

- 본 프로젝트는 슬라이드에 포함된 정보만을 기반으로 스크립트를 생성합니다.
슬라이드에 없는 내용을 추론하거나 확장하지 않도록 설계되어 있습니다.

- 수식이나 기호를 사용하지 않는 설명을 목표로 하기 때문에,
수학적 엄밀성이 중요한 강의에는 적합하지 않을 수 있습니다.

- 외부 API를 사용하므로 네트워크 환경 및 API 사용량에 따라
실행 시간과 비용이 달라질 수 있습니다.

- 본 프로젝트는 연구 및 실험 목적의 파이프라인으로,
대규모 강의 콘테츠 자동 생성을 위한 상용 시스템을 목표로 하지는 않습니다.
